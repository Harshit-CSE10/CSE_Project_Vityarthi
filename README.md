# CSE_Project_Vityarthi
# Local Business Inventory Manager

## Overview
A Python-based application to help small businesses manage stock and prevent stockouts.

âœ¨ Features
1. Inventory Management (CRUD Operations)

Add new products with details (name, SKU, category, price, quantity)
View all products in a formatted table
Update product information and stock levels
Delete products from inventory
Search products by name, SKU, or category
Track minimum stock levels for each product

2. Sales Tracking

Record sales transactions
Automatic inventory deduction upon sale
View sales history with product details
Generate sales summaries and analytics
Track revenue and units sold
Identify top-selling products

3. Low Stock Alerts

Automatic alerts when stock falls below minimum level
Critical alerts for out-of-stock items
View all active alerts in priority order
Auto-resolution of alerts when stock is replenished
Alert history tracking

4. Report Generation

Comprehensive inventory reports
Sales performance reports
Low stock and reorder recommendations
Category-wise analysis
Export reports to text files
