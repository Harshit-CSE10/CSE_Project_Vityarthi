# Local Business Inventory Manager

## Overview
A Python-based application to help small businesses manage stock and prevent stockouts.

## Features
- **Inventory CRUD:** Add, update, and delete products.
- **Sales Tracking:** Automatically deducts stock.
- **Alerts:** Warns when stock < 5 units.

## How to Run
1. Ensure Python 3 is installed.
2. Run the command: `python src/main.py`
